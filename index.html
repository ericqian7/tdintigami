TDINTigami

What is TDINTigami?

TDINTigami is inspired by Scorigami. It displays every unique touchdown-to-interception ratio by a quarterback in a given season since 1932.

Data Collection

Data from over 2,000 webpages on pro-football-reference.com was scraped using the BeautifulSoup Python package. Each quarterback listed has started at least one game during a season. The dataset includes 3,667 quarterbacks.
