TDINTigami

What is TDINTigami?

TDINTigami is inspired by Scorigami. TDINTigami displays every unique touchdown to interception ratio by a quarterback since 1932.

Data Collection

Data was collected from profootballreference.com. I couldn't find a data file containing the information I wanted so I wrote a python script to webscrape roughly 2000 webpages using the BeautifulSoup library. Each quarterback in TDINTigami had to have started at least started one game that season and have position be officially a quarterback. The data is stored in the igami.csv file.

Why 1932?

Unfourtanetly profootballreference doesn't have quarterback data from before 1932 and other sources were unreliable at best. Thankfully the quarterback was essentially non existant back then (there were only 6 uniqe players at the qb position from 1932-1940!) so most likely all qb stats are included.
